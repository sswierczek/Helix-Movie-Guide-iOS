# Setup

## Dependencies

Run ``` carthage update --platform iOS ``` to fetch dependencies.

## Config

1. Rename file Config_template.swift to Config.swift. 
2. Rename class Config_TEMPLATE to Config.
3. Put your Movie DB API key there.
